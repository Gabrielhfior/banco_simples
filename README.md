# Banco Simples

Projeto desenvolvido em Python para praticar lógica de programação.

## Funcionalidades

* Consulta de saldo
* Depósito
* Saque
* Histórico da última operação
* Meta financeira ao atingir R$1000

## Conceitos praticados

* Variáveis
* Condicionais (`if`, `elif`, `else`)
* Laços de repetição (`while`)
* Funções (`def`)
* Manipulação de valores numéricos
* Menus interativos no terminal

Projeto criado como parte dos meus estudos de programação em Python.
